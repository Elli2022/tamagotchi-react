# Tamagotchi React

Virtual Tamagotchi pet built with React and Vite — device-inspired UI, mood states, and care actions.

## Live Preview

<https://tamagotchi-react-elli2022.netlify.app>

**Repository:** [Elli2022/tamagotchi-react](https://github.com/Elli2022/tamagotchi-react)

## Screenshot

![Updated Tamagotchi UI](docs/tamagotchi-ui.png)

## Tech Stack

- React 18
- Vite 5
- Modern CSS animations and responsive, device-inspired UI

## Highlights

- Component-based React app with clear game loop and state handling
- Physical Tamagotchi-style shell with screen, controls, and status panel
- Mood transitions and action feedback

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
